1. Замените 1.xlsx на свой файл с организациями (название файла обязательно "1.xlxs", но мы над этим работаем)

2. Запустите main.exe

3. В открывшейся консоли введите количество тысяч посещений по которым будет произведена фильтрация сайтов организаций

4. После выполнение в этой папке появится файл "Ссылки.xlsx" - ваши лиды
